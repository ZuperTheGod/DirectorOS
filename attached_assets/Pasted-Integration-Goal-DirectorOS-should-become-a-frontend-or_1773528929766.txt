Integration Goal

DirectorOS should become a frontend orchestration layer that connects to local AI tools via APIs.

New architecture:

DirectorOS (Frontend + API Server)
        │
        ├── LM Studio → AI Director reasoning
        ├── ComfyUI → image generation workflows
        ├── Wan2 → video generation
        └── FFmpeg → timeline rendering


DirectorOS remains the creative interface, while external tools perform the heavy AI work.

1. LM Studio Integration (AI Director)

LM Studio runs LLMs locally and exposes an OpenAI-compatible API.

Typical endpoint:

http://localhost:1234/v1/chat/completions

Task for AI coder

Create a service module:

modules/llm/lmstudio.ts

Requirements

DirectorOS must:

send prompts to LM Studio

receive streaming responses

support structured output for scene generation

Example implementation outline
import axios from "axios";

export async function callLMStudio(messages) {
  const response = await axios.post(
    "http://localhost:1234/v1/chat/completions",
    {
      model: "local-model",
      messages,
      temperature: 0.7,
      stream: false
    }
  );

  return response.data.choices[0].message.content;
}

Update Director Endpoint

Modify:

/api/projects/:projectId/director/chat


Replace OpenAI call with:

LM Studio call

Director Tasks Using LM Studio

LM Studio should handle:

idea interpretation
scene planning
shot planning
prompt creation
video prompt optimization

2. ComfyUI Integration (Image Generation)

ComfyUI exposes a REST API for running workflows.

Default endpoint:

http://localhost:8188

Task for AI coder

Create a new service module:

modules/comfyui/comfyClient.ts

Function

Submit generation jobs to ComfyUI workflows.

Example request:

import axios from "axios";

export async function runComfyWorkflow(workflow) {
  const response = await axios.post(
    "http://localhost:8188/prompt",
    workflow
  );

  return response.data;
}

Image Generation Workflow

DirectorOS should send:

prompt
negative prompt
resolution
seed


to a ComfyUI workflow.

Example workflow type:

text_to_image.json

Backend Endpoint

Modify:

POST /api/shots/:shotId/generate-image


Instead of OpenAI image generation:

Build ComfyUI workflow JSON

Send workflow to ComfyUI

Wait for output image

Save image to project assets

4. FFmpeg Integration (Timeline Export)

Timeline rendering should use FFmpeg.

Task for AI coder

Create module:

modules/render/ffmpegRenderer.ts

Function

Combine clips into final video.

Example:

import { exec } from "child_process";

export function renderTimeline(clips, output) {
  const inputs = clips.map(c => `-i ${c}`).join(" ");
  
  const command = `
    ffmpeg ${inputs}
    -filter_complex concat=n=${clips.length}:v=1:a=0
    -y ${output}
  `;

  exec(command);
}

Endpoint
POST /api/projects/:id/export


Pipeline:

Timeline clips
↓
FFmpeg render
↓
Final MP4
